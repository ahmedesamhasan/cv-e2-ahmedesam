Description

A production-focused RESTful E-Commerce API with authentication, product management, carts, and orders. The project follows clean structure, testing, and containerized development.

Main Features

  • Secure authentication with Laravel Sanctum
  • Role-based access for admin and customer flows
  • Full CRUD for products, carts, and orders
  • Dockerized local environment for consistent setup
  • PHPUnit test coverage for critical backend flows
  • Postman-ready endpoint documentation
